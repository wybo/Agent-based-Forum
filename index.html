Agent-based Forum

by Wybo Wiersma

Agents posting on a forum, view source (download), available on Github, A-GPL-ed

Usage

  • Click Next generation to run step by step
  • Start/Pause simulation to toggle running it continuously
  • Old/New threads first to toggle the counter-factual of showing old threads to new users first (on the left)
  • Reset returns it to the start condition
